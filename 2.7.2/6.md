## match

```ruby
> text = '私の誕生日は1985年7月6日です'
=> "私の誕生日は1985年7月6日です"
> m = text.match(/(\d+)年(\d+)月(\d+)日/)
=> #<MatchData "1985年7月6日" 1:"1985" 2:"7" 3:"6">
> m[0]
=> "1985年7月6日"
> m[1]
=> "1985"
> m[2, 2]
=> ["7", "6"]
> n = text.match(/(?<year>\d+)年(?<month>\d+)月(?<day>\d+)日/)
=> #<MatchData "1985年7月6日" year:"1985" month:"7" day:"6">
> n[0]
=> "1985年7月6日"
> n[1]
=> "1985"
> n[:year]
=> "1985"
> n[2]
=> "7"
> n[:month]
=> "7"
> n[:day]
=> "6"

> /(?<year>\d+)年(?<month>\d+)月(?<day>\d+)日/ =~ text
=> 6
> year
=> "1985"
> month
=> "7"
> day
=> "6"
```

## gsub

```ruby
> text = '123,456-789'
=> "123,456-789"
> hash = { ',' => ':', '-' => '/' }
=> {","=>":", "-"=>"/"}
> text.gsub(/,|-/, hash)
=> "123:456/789"
```
